# Wedding Website

A static GitHub Pages website for our Rome wedding celebration.
<!-- test push: 2026-03-08 -->

## View the Website

Once deployed, this website will be available at: `https://bereasonable.github.io/wedding/`

## Features

- Responsive design that works on all devices
- Wedding details (date, venue, celebration info)
- RSVP information
- Travel and accommodations section
- Clean, elegant design with purple/lavender theme

## Deployment

This site is automatically deployed to GitHub Pages from the repository.

## Local Development

To view the site locally, simply open `index.html` in your web browser.
