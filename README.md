# Wedding Website

A static GitHub Pages website for our Rome wedding celebration.
<!-- test push: 2026-03-08 -->

## View the Website

Once deployed, this website will be available at: `https://bereasonable.github.io/wedding/`

## Features

- Responsive design that works on all devices
- Wedding details (date, venue, celebration info)
- RSVP information
- Travel and accommodations section
- Clean, elegant design with purple/lavender theme

## Deployment

This site is automatically deployed to GitHub Pages from the repository.

## Local Development

```bash
npm install       # first time only
npm run dev       # starts dev server at http://localhost:3000 with live reload
```

Edit `index.html`, `styles.css`, or any other files — the browser will update automatically.

## Deploying

When the changes look good locally, commit and push to `main`:

```bash
git add .
git commit -m "your message"
git push
```

GitHub Pages will automatically serve the updated content (usually within a minute).
